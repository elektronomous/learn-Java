1. What keyword is used in Java to indicate that the value of 
   a variable will not change (i.e the variable is actually 
   a constant).

   = final

2. What is the Java convention for naming constants?

   = Using uppercase and underscore to seperate if the names
     are more than one word.

3. Name one advante of creating names for constants.
   
   = Easier to read

4. Why is TWENTY_FOUR not a good name for the constant 24?

   = The name of the constants should be the meaning of the constants
     not the value of the constants.
