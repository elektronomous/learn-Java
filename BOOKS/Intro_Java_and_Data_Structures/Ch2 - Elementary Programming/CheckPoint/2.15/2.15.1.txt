Which of these statements are true?
        a. Any expression can be used as a statement.
        b. The expression x++ can be used as a statement.
        c. The statement x = x + 5 is also an expression.
        d. The statement x = y = x = 0 is illegal.

(answer):

a. Any expression can be used as a statement.       (true)
b. The expression x++ can be used as a statement.   (true)
c. The statement x = x + 5 is also an expression.   (true)
d. The statement x = y = x = 0 is illegal.          (false)
