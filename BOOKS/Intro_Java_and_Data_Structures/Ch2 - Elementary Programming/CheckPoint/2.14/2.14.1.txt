Show the output of the following code:

        double a = 6.5;
        a += a + 1;
        System.out.println(a);
        a = 6;
        a /= 2;
        System.out.println(a);

(answer):

        double a = 6.5;
        a += a + 1;
        System.out.println(a);      // => 14.0
        a = 6;
        a /= 2;
        System.out.println(a);      // => 3.0
