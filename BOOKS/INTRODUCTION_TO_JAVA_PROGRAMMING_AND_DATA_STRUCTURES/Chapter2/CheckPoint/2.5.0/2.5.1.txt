= Identify and fix the errors in the following code:

    public class Test {
        public static void main(String[] args) {
            int i = k + 2;
            System.out.println(i);
        }
    }


(answer): It should declare and initialze the "k" variable before it's used.

    public class Test {
        public static void main(String[] args) {
            int k = 10;
            int i = k + 2;
            System.out.println(i);
        }
    }
